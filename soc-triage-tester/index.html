Connection Troubleshooting

It looks like your browser can't connect to Ollama. Here's how to fix it:

1
Verify Ollama is running

Open a terminal and run:

ollama serve

Or if it's already running, check its status with:

ollama list
2
Check if Ollama is accessible

Open a new tab and visit: http://localhost:11434

If you see a message like "Ollama is running", then Ollama is working but there might be a CORS issue.

3
Fix CORS issues

Ollama might be blocking requests from the browser. Try running Ollama with CORS enabled:

OLLAMA_ORIGINS="*" ollama serve

Or set the environment variable first:

export OLLAMA_ORIGINS="*"
ollama serve
4
Try a different approach

If you're still having issues, you can run a local proxy server:

npm install -g local-cors-proxy
lcp --proxyUrl http://localhost:11434

Then use http://localhost:8010/proxy as your API endpoint.
